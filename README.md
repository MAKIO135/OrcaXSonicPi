# Orca X Sonic Pi

Démo d'[Orca](https://github.com/hundredrabbits/Orca) avec [Sonic Pi](https://sonic-pi.net/)

Se base sur l'[opérateur OSC](https://github.com/hundredrabbits/Orca#osc) d'Orca `=`

#### Configuration Orca
- UDP: Orca (49160)
- OSC: Sonic Pi (4559)
